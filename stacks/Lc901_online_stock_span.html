LC 901 — Online Stock Span

Medium Time O(n) amortized Space O(n)
class StockSpanner: def __init__(self): self.stack = [] # (price, span) def next(self, price): span = 1 while self.stack and self.stack[-1][0] <= price: span += self.stack.pop()[1] self.stack.append((price, span)) return span
