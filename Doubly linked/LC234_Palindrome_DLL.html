LeetCode 234 — Palindrome Linked List (Doubly Linked List)

Easy ⏱ O(n) 💾 O(1) ⇄ DLL

DLL superpower: right moves backwards via .prev — no reversal needed!
Code
def isPalindrome(head):
if not head or not head.next:
return True
# Find the tail
tail = head
while tail.next:
tail = tail.next
# Two-pointer from both ends
left, right = head, tail
while left != right and left.prev != right:
if left.val != right.val:
return False
left = left.next
right = right.prev # ← DLL superpower
return True
